# pager-cli-client

```sh
git clone https://github.com/glebchanskiy/pager-cli-client
cd pager-cli-client
poetry install

# user must be registered on the server
# you can use: 
# glebchanskiy
# 1111
poetry run client {name}
```

## Usage
```sh
@show - показать последние 10 сообщений
@show 50 - показать последние 50 сообщений
@help - usage
# to send a message, simply enter the text
```
